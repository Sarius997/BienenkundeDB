Installation

Es gibt 3 mögliche Varianten BienenkundeDB zu nutzen:

nur BienenkundeDB:

1. Laden Sie "BienenkundeDB.jar" herunter
2. Das Programm kann sofort genutzt werden

BienenkundeDB Installer:

1. Laden Sie "BienenkundeDB Installer.exe" herunter
2. Führen Sie den Installer aus
3. BienenkundeDB kann nun über die Verknüpfung zum Launcher gestartet werden

BienenkundeDB mit Launcher:

1. Laden Sie "BienenkundeDB.jar" und "BienenkundeDB Launcher.exe" herunter
2. Starten Sie BienenkundeDB.jar
3. Klicken Sie unter Hilfe > Info auf "Speicherort öffnen"
4. Beenden Sie BienenkundeDB.jar
5. Verschieben Sie beide heruntergeladenen Dateien in dem vorhin geöffneten Ordner
6. Erstellen Sie eine Desktopverknüpfung zur "BienenkundeDB Launcher.exe" (kann auch BienenkundeDB.Launcher.exe heißen)

Unterschiede der Verschiedenen Anwendungsweißen:

Anwendungsvariante Installationsaufwand Autoupdate Feature
nur .jar gering (sofort verwendbar) Neustart muss vom Anwender durchgeführt und Dateien umbenannt werden
Installer gering (nach Installation sofort verwendbar) Voll funktionsfähig
manuelle Installation hoch (Dateien müssen in bestimmte Ordner verschoben werden, mehrere Schritte notwendig) Voll funktionsfähig
